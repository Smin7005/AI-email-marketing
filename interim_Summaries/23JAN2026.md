# 2. For Inngest to work on Vercel in production, you need:
  1. INNGEST_EVENT_KEY - to send events to Inngest  
  2. INNGEST_SIGNING_KEY - for Inngest to verify webhook requests to your app  
  The user's .env.local has placeholder values for these (ZC-... and signkey-...) which are not real keys.                                         
  The user needs to:
  1. Go to https://app.inngest.com/ and create an account/project
  2. Get the real INNGEST_EVENT_KEY and INNGEST_SIGNING_KEY from the Inngest dashboard
  3. Add these to Vercel environment variables
